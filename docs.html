项目文档

这里整理了在线演示平台的结构、能力范围、访问方式与适用场景，方便用于展示或提交申请材料。

项目定位

本项目是一个面向嵌入式软件研发的 Agent / AI 在线演示页面，重点展示超大型自动化编码、自动化测试构建、调试版本分析和团队共用场景。它以静态 Web 页面形式提供，适合作为能力展示入口和项目说明页。

在线访问

当前服务通过静态 Web 页面运行，可通过以下路径访问：

http://127.0.0.1:8080/

如果当前主机已开放外部访问并映射 8080 端口，也可使用对应公网 IP 或域名访问。

核心展示内容

自动化编码

展示多模块嵌入式工程中的批量代码生成、补全与重构能力。

自动化测试

展示基于代码与需求文档自动生成测试用例、脚本与回归验证内容的能力。

调试与排障

展示日志分析、问题定位、调试版本迭代与修复建议输出能力。
